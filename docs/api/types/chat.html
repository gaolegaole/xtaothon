Chat()¶

class telethon.tl.types.chat¶

A chat.

参数:
id (int):

唯一标识符。

title (str):

标题

username (str, 可选):

如果未配置，则返回 None 。

participants_count (int):

群成员数

date (int):

可能是创建日期？

slowmode_enabled (bool):

是否开启慢速模式，如果无法获取则返回 None 。

creator (bool):

如果当前客户端登录用户是创建者，则返回 True 。

kicked (bool):

如果当前客户端登录用户被踢出，则返回 True 。

left (bool):

如果当前客户端登录用户已退出，则返回 True 。

call_active (bool):

如果群语音已激活，则返回 True 。

call_not_empty (bool):

如果群语音有成员，则返回 True 。
