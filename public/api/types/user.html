User()¶

class telethon.tl.types.user¶

A Telegram user or bot.

参数:
id (int):

唯一标识符。

is_self (bool):

如果是当前客户端登录用户，则返回 True 。

bot (bool):

如果是机器人，则返回 True 。

verified (bool):

如果是官方认证，则返回 True 。

restricted (bool):

如果是收到限制，则返回 True 。

fake (bool):

如果是被标记为冒充，则返回 True 。

first_name (str):

first name

last_name (str, 可选):

如果未配置，则返回 None 。

username (str, 可选):

如果未配置，则返回 None 。

phone (str, 可选):

如果无法读取，则返回 None 。

lang_code (str, 可选):

如果无法读取，则返回 None 。
